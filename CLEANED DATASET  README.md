# Cleaned Global YouTube Statistics Dataset

## Overview

This repository contains a cleaned and preprocessed dataset of global YouTube statistics. The dataset is structured to support accurate analysis, visualization, and reporting.

## Data Cleaning Process

The following steps were performed to prepare the dataset:

1. Removed null and missing values
2. Eliminated duplicate records
3. Converted numeric columns (views, subscribers, uploads) from text to numeric format
4. Removed special characters such as commas and symbols
5. Standardized column formats and naming conventions
6. Validated data types for analytical consistency

## Dataset Description

The dataset includes the following key attributes:

* Video views
* Subscriber count
* Upload count
* Channel type
* Category
* Ranking metrics

## Steps to Use the Dataset

1. Download the CSV file from this repository
2. Open the dataset in a tool such as Excel, Power BI, or Python
3. Verify data types and column formats
4. Perform analysis or build visualizations as required

## Applications

This dataset can be used for:

* Data analysis and exploration
* Dashboard creation in Power BI or Tableau
* SQL querying and reporting
* Machine learning practice

## Conclusion

The dataset has been cleaned and structured to ensure reliability and ease of use for data analytics tasks.
